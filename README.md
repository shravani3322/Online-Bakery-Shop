# 🍰 Online Bakery Shop

## 📝 Description
Online Bakery Shop is a mini e-commerce project built using PHP.  
It allows users to browse bakery products and place orders online.

## 🛠️ Technologies Used
- HTML  
- CSS  
- PHP  
- Bootstrap

## ✨ Features
- Product listing with images  
- Add to cart and checkout  
- Simple order form  
- Responsive layout
